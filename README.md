# 🏥 Healthcare Data Warehouse & Power BI and Readmission Prediction Project (FULL PROJECT)

This repository contains the files and documentation for a comprehensive healthcare data project, including a Data Warehouse, Power BI dashboards, and a Machine Learning model for patient readmission prediction.

## 📂 Repository Contents

Here is a detailed breakdown of the files included in this project:

* **[📁 research](./research)**
  A folder containing the research papers, references, and foundational resources utilized throughout the project.

* **[🗄️ Data Warehouse (ZIP)](./Data%20Warehouse.zip)**
  A compressed file containing the schema, structure, and source files of the project's Data Warehouse.

* **[📄 Healthcare DW Documentation](./Healthcare_DW_Documentation.docx)**
  The official documentation detailing the design, architecture, and implementation of the Data Warehouse.

* **[📊 Healthcare Dashboards Full Report](./Healthcare_Dashboards_Full_Report_Document.docx)**
  A comprehensive report explaining the data visualizations, KPIs, and analytical insights from the Power BI dashboards.

* **[📽️ Healthcare Project Presentation](./Healthcare_Project_Presentation%20(2).pptx)**
  The PowerPoint presentation used for the project defense and overview.

* **[💻 Prediction Model (Jupyter Notebook)](./Model.ipynb)**
  The Jupyter Notebook containing the Machine Learning code used for data preprocessing, training, and predicting patient readmissions.

* **[📈 Power BI Graduation Project](./Pbip.graduation_project.rar)**
  The original Power BI project files (`.pbip` / `.rar`) containing the interactive dashboard designs and datasets.

* **[📄 Readmission Model Documentation](./Readmission_Model_Documentation.docx)**
  The documentation explaining the methodology, feature engineering, and evaluation metrics of the patient readmission prediction model.

* **[📄 Power Bi Project & Raw Data](https://drive.google.com/drive/folders/1G0stgMEAIWOcDkEuv_UNu-PsWmkUMyBG?usp=drive_link)**
 The link contains raw data, final dashboards and modeling.
